#Amazon Homepage Clone

A frontend clone of the Amazon homepage built using HTML and CSS while learning web development.

#Features

- Responsive homepage layout
- Navigation bar
- Search bar design
- Product sections
- Footer section

#Technologies Used

- HTML5
- CSS3

#Purpose

This project was created for practice to improve frontend development skills such as layout design, styling, and responsive structure.
